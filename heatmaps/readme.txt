WARNING!!!! Heatmap generation will consume alot of cpu for a long time while generating your heatmaps

300k kills on one map will take ~10min to generate on one 2.4Ghz core cpu.

Install:
	- change user, password, database settings in generate.php
	- change path to where you want your heatmaps to be put, in
	  generate.php

report problems to hlx forums or irc
